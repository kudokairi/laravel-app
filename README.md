# SNSHU
### アプリ概要
Twitterのようにつぶやきたいことをつぶやけるアプリです。
タグを有効活用することで同じ考えや趣味の人をみつけていくことができます。
見る専用の人でも登録なしで利用可能ですが登録することで気になった人はフォローできたり、気に入った投稿をいいねできたりします。

### 機能一覧
- 会員登録
- ログイン・ログアウト
- 記事の作成・表示・更新・削除（CRUD処理）
- 部分検索
- マイページ
- フォロー
- タグ
- いいね
- レスポンシブ対応
